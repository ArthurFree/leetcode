示例 2：
```
输入：["aa","bb","ab","ba"]
输出：4
解释：4 组 ["aa"]，["bb"]，["ab"]，["ba"]
```

示例 3：
```
输入：["abc","acb","bac","bca","cab","cba"]
输出：3
解释：3 组 ["abc","cba"]，["acb","bca"]，["bac","cab"]
```

示例 4：
```
输入：["abcd","cdab","adcb","cbad"]
输出：1
解释：1 组 ["abcd","cdab","adcb","cbad"]
```

提示：

- 1 <= A.length <= 1000
- 1 <= A[i].length <= 20
- 所有 A[i] 都具有相同的长度。
- 所有 A[i] 都只由小写字母组成。
